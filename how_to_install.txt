	Αρχικά για την εγκατάσταση της εφαρμογής απαιτείται η εγκατάσταση του mysql server για τη βάση δεδομένων.Δημιουργούμε σε αυτόν χρήστη dbs_22, με κωδικό dbs_22_password και μια βάση δεδομένων με όνομα dbs_22_tst. Για τη δημιουργία των πινάκων χρησιμοποιούμε το αρχείο dbs_22.sql. Για την εισαγωγή ψεύτικων δεδομένων για δοκιμές χρησιμοποοιούμε το αρχείο fake.sql.
	Για να λειτουργήσει το backend πρέπει να εγκαταστήσουμε αρχικά τον χειριστή πακέτων npm και την πλατφόρμα nodejs.Με τη χρήση του npm κατεβάζουμε τα πακέτα express,cors.Η λειτουργία του backend ξεκινάει δίνοντας την εντολή node index.js.
	Για το frontend απαιτείται λήψη της angular(με χρήση του npm), δημιουργία νέου project(ng new frontend) και στον φάκελο που δημιουργείται, στον υποφάκελο src/app τα αρχεία πρέπει να αντικατασταθούν απο τα ομώνυμά τους που δίνονται..Η λειτουργία του frontend ξεκινάει δίνοντας την εντολή ng serve μέσα στο φάκελο.
